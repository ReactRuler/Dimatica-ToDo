## Project Summary

A full-stack Todo Task application with React + TypeScript frontend and Express.js + TypeScript backend, using PostgreSQL for data persistence. Developed and tested on Ubuntu OS with comprehensive testing coverage.

### Frontend

- Built with React 18+ and TypeScript.
- Uses Ant Design for UI components and icons.
- Supports creating, updating, deleting todos and subtasks.
- Communicates with backend API for data persistence.
- Unicode-safe string handling with `runes2`.
- Tested with Jest and React Testing Library.

---

### Backend

- Built with Express.js and TypeScript.
- Provides RESTful API endpoints for todos and subtasks.
- Uses PostgreSQL for data storage.
- Environment configuration via `.env` file.
- CORS enabled for cross-origin requests.
- Tested with Jest and Supertest for API integration.

---

### Development & Deployment

- Runs on Ubuntu environment.
- Backend listens on configurable port (default 3000).
- Frontend runs on development server (default port 3000).
- Uses environment variables for configuration.
- Includes comprehensive test suites covering success and failure cases.

### Technologies Used

- **Languages & Frameworks:** TypeScript, React, Express.js, Node.js
- **Database:** PostgreSQL
- **UI Library:** Ant Design
- **Testing:** Jest, React Testing Library, Supertest
- **OS:** Ubuntu
- **Others:** dotenv, cors, runes2
